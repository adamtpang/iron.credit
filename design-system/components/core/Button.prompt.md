The Iron button — one orange primary per view for the key action.

```jsx
<Button variant="primary" size="lg">Join the waitlist</Button>
<Button variant="secondary">View proof of reserves</Button>
<Button variant="ghost">Learn more</Button>
```

Variants: `primary` (bitcoin orange), `secondary` (steel outline), `ghost`. Sizes `sm`/`md`/`lg`, plus `full`, `leadingIcon`, `trailingIcon`, `disabled`.
