A mono, engraved uppercase pill — status and LTV-health signalling.

```jsx
<Badge tone="safe" dot>Healthy</Badge>
<Badge tone="caution">Approaching limit</Badge>
<Badge tone="danger" dot>Liquidation risk</Badge>
<Badge tone="orange">Non-custodial</Badge>
```

Tones: `neutral`, `safe`, `caution`, `danger`, `orange`. Add `dot` for a leading indicator.
